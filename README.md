# Инвестиционный Портфель

Современное веб-приложение для управления инвестиционным портфелем, созданное с использованием React и TypeScript.

## Функциональность

- **Автоматическая загрузка данных**: Все параметры ценных бумаг загружаются автоматически по тикеру
  - Текущая цена
  - Изменение цены и процента
  - Дивидендная доходность
  - Название компании
- **Управление ценными бумагами**: Добавляйте акции, облигации и ETF в свой портфель
- **Управление недвижимостью**: Отслеживайте инвестиции в недвижимость
- **Автоматическое обновление цен**: Цены обновляются автоматически при загрузке и по запросу
- **Отображение текущих цен**: Видите актуальную стоимость всех активов в реальном времени
- **Отслеживание изменений**: Автоматический расчет изменения цены и процента (цветовая индикация)
- **Ожидаемые дивиденды**: Расчет ожидаемых дивидендов от ценных бумаг с учетом частоты выплат
- **Арендный доход**: Расчет ожидаемого дохода от недвижимости
- **Сохранение данных**: Портфель автоматически сохраняется в локальном хранилище браузера

## Технологии

- React 18
- TypeScript
- Vite
- CSS3

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Откройте браузер и перейдите по адресу, указанному в терминале (обычно http://localhost:5173)

## Сборка для production

```bash
npm run build
```

## Использование

1. **Добавление ценных бумаг:**
   - Нажмите кнопку "Добавить ценную бумагу"
   - Введите тикер (например: SBER, GAZP, AAPL)
   - После выхода из поля ввода данные загрузятся автоматически
   - Укажите количество и тип бумаги
   - Нажмите "Добавить"

2. **Добавление недвижимости:**
   - Нажмите кнопку "Добавить недвижимость"
   - Заполните форму с информацией о недвижимости

3. **Обновление цен:**
   - Цены автоматически обновляются при загрузке страницы

4. **Просмотр портфеля:**
   - Общая стоимость портфеля и ожидаемый доход отображаются в верхней части
   - Каждая карточка показывает актуальную цену, изменение и дивиденды
   - Удаляйте активы, нажав кнопку "×" на карточке

5. **Поддерживаемые тикеры:**
   - Российские акции: SBER, GAZP, LKOH, YNDX и др.
   - Американские акции: AAPL, GOOGL, MSFT и др.
   - При использовании Yahoo Finance API поддерживаются все публичные тикеры

Все данные автоматически сохраняются в браузере и будут доступны при следующем посещении.
