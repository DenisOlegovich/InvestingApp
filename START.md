# 🚀 Как запустить приложение

## Терминал 1: Backend

```bash
cd /Users/denis/Desktop/Програмирование/App/backend
npm start
```

## Терминал 2: Frontend  

```bash
cd /Users/denis/Desktop/Програмирование/App
npm run dev
```

## Откройте браузер

Перейдите на `http://localhost:5173`

## Первый вход

1. **Нажмите "Зарегистрироваться"**
2. Введите:
   - Имя: `Ваше имя`
   - Email: `your@email.com`
   - Пароль: `password` (минимум 6 символов)
3. Нажмите "Зарегистрироваться"
4. Вы автоматически войдете в систему
