# 🚀 Как запустить приложение

## Терминал 1: Backend

```bash
cd /Users/denis/Desktop/Програмирование/App/backend
npm start
```

## Терминал 2: Frontend  

```bash
cd /Users/denis/Desktop/Програмирование/App
npm run dev
```

## Откройте браузер

Перейдите на `http://localhost:5173`

## Первый вход

1. **Нажмите "Зарегистрироваться"**
2. Введите:
   - Имя: `Ваше имя`
   - Email: `your@email.com`
   - Пароль: `password` (минимум 6 символов)
3. Нажмите "Зарегистрироваться"
4. Вы автоматически войдете в систему

## Или используйте тестовый аккаунт:

## 📊 Возможности приложения

- **Дашборд, графики, аллокация** — обзор портфеля и структура активов
- **История сделок** — учёт операций покупки/продажи
- **Налоговый отчёт** — упрощённый расчёт на основе сделок
- **Экспорт** — JSON, Excel, PDF
- **Импорт** — загрузка бумаг из CSV
- **PWA** — установка как приложение, офлайн
- **Тёмная/светлая тема**
- **Горячие клавиши** — быстрый переход по вкладкам (Ctrl+1–5)
- **Восстановление пароля** — сброс через email
- **Калькуляторы** — сложный процент, вычет по ИИС
- **Цели, watchlist, чек-листы, заметки**

## 📊 Что хранится на сервере:

- Ваши ценные бумаги
- Недвижимость
- Депозиты
- Криптовалюты

Все данные привязаны к вашему аккаунту и защищены!
